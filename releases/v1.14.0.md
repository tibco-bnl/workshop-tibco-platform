# Release Notes - v1.14.0

**Release Date:** February 27, 2026  
**TIBCO Platform Control Plane Version:** 1.14.0

## Overview

This release contains documentation and scripts aligned with **TIBCO Platform Control Plane version 1.14.0** for setting up TIBCO Platform sandbox environments across various deployment platforms.

## What's Included

### Documentation
- Complete setup guides for multiple base VM platforms (WSL, Ubuntu, AWS, Azure)
- Kubernetes deployment guides (Minikube, k3d, kind, Docker Desktop)
- TIBCO Platform Control Plane and Data Plane installation instructions
- Platform configuration and customization guides
- Recipe guides for common deployment scenarios
- Developer Hub setup documentation

### Scripts
- Certificate generation and management
- Ingress configuration automation
- Database setup scripts
- Helper scripts for platform deployment

## Supported Platforms

### Base VM Options
- Docker Desktop for WSL with Kubernetes
- Ubuntu on WSL
- Ubuntu on AWS
- Ubuntu on Azure

### Kubernetes Options
- Minikube
- k3d
- kind
- Docker Desktop Kubernetes

## Key Features

- ✅ Multiple deployment platform support
- ✅ Comprehensive Control Plane and Data Plane setup
- ✅ Platform configuration guides
- ✅ Recipe-based deployment scenarios
- ✅ Developer Hub integration
- ✅ Certificate and ingress management
- ✅ Database configuration support

## Compatibility

- **TIBCO Platform Control Plane:** v1.14.0
- **Kubernetes:** 1.24+
- **Helm:** 3.8+
